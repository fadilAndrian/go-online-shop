# Go Online Shop API

Backend REST API sederhana untuk simulasi e-commerce.

## Tech Stack
- Go
- Gin (HTTP framework)
- GORM (ORM)
- MySQL

## Fitur
- Authentication (JWT) [on progress]
- CRUD Product
- Order flow sederhana [on progress]
- Validation & error handling 

## Cara Jalankan
1. Clone repository:
   git clone https://github.com/fadilAndrian/go-online-shop.git
2. Masuk folder project:
   cd go-online-shop
3. Install dependencies:
   go mod tidy
4. Masuk ke /cmd/app:
   cd cmd/app
5. Jalankan:
   go run .
